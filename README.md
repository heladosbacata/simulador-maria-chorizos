# simulador-maria-chorizos

Simulador de rentabilidad para la franquicia María Chorizos. Landing de una sola página con cálculo en tiempo real de ingresos, costos, utilidad y ROI.

## Despliegue

- **Local:** `python3 -m http.server 8000` y abre http://localhost:8000
- **Vercel:** Conectar este repo en [vercel.com](https://vercel.com) para deploy automático.

Ver `GUIA-DESPLIEGUE.md` para pasos detallados.
